Input type: Text

name: Specifies the name of the input field, used to identify it when submitting form data.
value: Sets the initial value of the input field.
maxlength: Limits the maximum number of characters the user can input.
placeholder: Displays a hint or guide text inside the text field (disappears when the user starts typing).
required: Makes the field mandatory before form submission.
readonly: Prevents the user from editing the field.
disabled: Disables the field, making it uneditable and unsubmitable.
pattern: Specifies a regex pattern the input must match.
autocomplete: Allows browsers to suggest saved inputs (e.g., on, off).
